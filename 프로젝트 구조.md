# 🏭 AI 품질 관리 시스템

중학생을 위한 Python 교육용 프로젝트

## 📁 프로젝트 구조

```
qc_project/
│
├── qc_main.ipynb          # 🎯 메인 프로그램 (학생들이 실행)
│
├── utils/                 # 🛠️ 도구 모음 폴더
│   ├── __init__.py       # 패키지 초기화
│   ├── logger.py         # 결함 기록 도구
│   ├── analyzer.py       # 결함 분석 도구
│   ├── display.py        # 화면 표시 도구
│   └── microbit.py       # 마이크로비트 제어 도구
│
├── data/                  # 📂 데이터 폴더
│   ├── test.mp4          # 테스트 영상
│   └── background.jpg    # 배경 이미지
│
└── gc10-det_openvino_model/  # 🤖 AI 모델 폴더
```

## 🚀 사용 방법

### 1단계: 환경 준비
```bash
# 필요한 라이브러리 설치
pip install opencv-python ultralytics pyserial
```

### 2단계: 파일 준비
1. `qc_main.ipynb` 파일을 주피터 노트북으로 엽니다
2. `utils` 폴더에 모든 모듈 파일이 있는지 확인합니다
3. `data` 폴더에 영상과 배경 이미지가 있는지 확인합니다

### 3단계: 설정 변경
`qc_main.ipynb`의 2단계에서 설정을 원하는 대로 변경합니다:
```python
USE_MICROBIT = False  # 마이크로비트 사용 여부
MODEL_PATH = "..."    # AI 모델 경로
VIDEO_PATH = "..."    # 영상 파일 경로
```

### 4단계: 실행
주피터 노트북에서 셀을 순서대로 실행합니다!

## 📚 각 모듈의 역할

### 🎯 qc_main.ipynb (메인 프로그램)
- 전체 시스템을 제어하는 메인 프로그램
- 학생들이 직접 코딩하고 수정하는 부분
- 간단하고 이해하기 쉽게 작성됨

### 📝 utils/logger.py
- 결함 정보를 CSV 파일에 저장
- 시간, 결함 종류, 위치 등을 기록

### 🔍 utils/analyzer.py
- AI가 찾은 결함을 분석
- 합격(PASS) / 불합격(NG) 판정
- 5가지 결함 종류 구분

### 🖼️ utils/display.py
- 검사 결과를 화면에 예쁘게 표시
- 시간, 결과, 결함 개수 등 정보 표시

### 🤖 utils/microbit.py
- 마이크로비트와 통신
- 검사 결과를 마이크로비트로 전송

## 🔧 문제 해결

### Q: 마이크로비트가 연결되지 않아요
A: `USE_MICROBIT = False`로 설정하고 실행하세요

### Q: 영상이 실행되지 않아요
A: `VIDEO_PATH`가 올바른지 확인하세요

### Q: CSV 파일에 기록이 안 돼요
A: 엑셀에서 기존 CSV 파일을 닫고 다시 실행하세요
